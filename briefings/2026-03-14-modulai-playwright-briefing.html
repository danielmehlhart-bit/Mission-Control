Was ist Playwright?
Browser-Automatisierung von Microsoft — kostenlos
  • 🏢
    Von Microsoft entwickelt Open Source, aktiv weiterentwickelt, riesige Community — keine Vendor-Abhängigkeit
  • 💰
    Komplett kostenlos Keine Abo-Kosten, keine versteckten Gebühren — läuft auf deinem Pi zuhause
  • 🖱️
    Steuert echte Browser Kein Fake — Playwright öffnet Chromium und klickt wirklich durch deine App, genauso wie ein echter User
✅ Gratis ✅ Kein Abo ✅ Keine Cloud notwendig 🔓 Open Source
🏗️
Was wir konkret bauen
Dein automatischer QA-Agent für ModulAI — läuft auf dem Pi
  • 🍓
    Läuft auf deinem Raspberry Pi 5 Keine Cloud, keine monatlichen Kosten — dein Pi arbeitet nachts während du schläfst
  • ⏰
    Startet automatisch um 02:00 Uhr Ein Cron-Job weckt Playwright jede Nacht — kein manuelles Starten nötig
  • 👻
    Headless Browser — unsichtbar im Hintergrund Playwright öffnet Chromium ohne Fenster und klickt sich durch ModulAI — vollautomatisch

Was Playwright nicht kann
Ehrlichkeit über Grenzen — damit Erwartungen stimmen
  • 🚫
    Kein echtes iOS Safari Playwright simuliert mobile Viewports — aber echter Safari auf dem iPhone hat eigene Bugs, die wir so nicht finden. Für Safari-spezifische Issues braucht es echtes Gerät.
  • 🚫
    Kein Performance-Testing Playwright prüft Funktion — nicht Ladezeiten, Core Web Vitals oder Server-Antwortzeiten. Dafür gibt's andere Tools (z.B. Lighthouse).
  • 🚫
    Kein visuelles Design-Review Playwright sieht nicht ob ein Button verschoben ist oder Farben falsch sind — nur ob Elemente vorhanden und klickbar sind.

Nächste Schritte
4 Schritte bis zum laufenden QA-System
  • 1
    Playwright auf dem Pi installieren npm install + Chromium-Binary — einmaliger Setup, danach läuft es
  • 2
    Test-Suite schreiben Login-Flow, Projekt anlegen, Tenant-Wechsel — die wichtigsten Kernfunktionen zuerst
  • 3
    Cron einrichten (02:00 Uhr täglich) Ein Befehl in der Crontab des Pi — danach startet alles automatisch
  • 4
    Telegram-Report konfigurieren Bot-Token + Chat-ID — du kriegst Ergebnis direkt aufs Handy
